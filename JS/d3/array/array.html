统计学相关

d3中array的计算方法，会忽略数组中不可识别的数据类型或者值，如null、undefined、null。

d3.min(iterable[,accessor])

接收一个迭代类型数据，还可以选择性传入一个数据访问器的函数，在里面可以对值进行操作。

d3.max(iterable[,accessor])

返回最大值

d3.extent(iterable[,accessor])

返回最大和最小值的数组

d3.sum(iterable[,accessor])

返回求和结果

d3.mean(iterable[,accessor])

平均值

d3.median(iterable[,accessor])

去中位数，奇数个参数返回大小关系中间值的数，偶数个参数返回大小关系中间值两个数的平均值，例如：d3.median([-99,20,-10,-50]) => 获取中卫大小为-50、-10 => 获取平均值-30

d3.quantile(iterable,p[,accessor])

求得四分位数，p是0到1之间的数。计算方法是按照p的值获取数组顺序对应位数的值，如果不在整位，则计算返回内值按照占比的平均值，例如：d3.quantile([-99,20,-10,-50], 1/3) => 按照位数0、1/3、2/3、1 => 返回20。再例如：d3.quantile([-100,20,-10,-50,50], 0.9) => -50是0.75,50是1，相当于在-50~50取得3/5的进度 => 返回10。可以按照线性折线图取值来判断。

d3.variance(iterable[,accessor])

方差，等于各个数据与其算术平均数的差的平方和的平均值。方差越大，数值越分散

d3.deviation(iterable[,accessor])

标准偏差，计算方法是将各项与平均值的差的平方和，求得平均值后开方，所以标准偏差也是方法的开方值。用于反应一个数据集的离散程度。

数据筛选

d3.scan(iterable[,comparator])

遍历数据，接收一个用于比较的函数，返回true就会把当前元素和下一个的元素进行比较，返回false把上一次的结果与下一次进行比较，最后返回符合条件的下表index。

d3.bisect(iterable,x[,lo[,hi]])

将x按顺序插入iterable数组对应的位置，如果x已经在数组中存在，那么插入下标在已存在元素下标之后，可以通过传入lo和hi截取这两个位置之间的数组进行判断，最后返回如果插入元素对应的索引。需要注意的是，应该对一个有序的数组进行操作。

d3.bisectLeft(iterable,x[,lo[,hi]])

功能同上，也是对插入元素排序，只不过遇到已存在的情况，返回的插入位置在已存在元素下标之前。

d3.bisectRight(iterable,x[,lo[,hi]])

与d3.bisect()完全一致。

d3.bisector(accessor|comparator)

一个按序插入处理器，接收一个数据处理函数或者比较函数，返回一个处理器对象bisector。处理器对象调用bisector.left和bisector.right方法就等于上面的定位方法d3.bisectLeft和d3.bisectRight

d3.quickselect(array, k, left = 0, right = array.length - 1, compare = ascending)

。

d3.ascending(a, b)

比较a、b两个数的大、小、相等关系，返回1、-1或0。这不是array的方法，但是属于d3.array很多方法中的基础函数(传入函数默认值)

d3.descending(a, b)

比较a、b两个数的大、小、相等关系，返回-1、1或0。
