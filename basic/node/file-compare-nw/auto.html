录入差异文件列表 本地文件夹比对结果与录入差异文件列表的匹配结果
    使用说明：

    1.点击选择文件，左侧选择作为基准的文件夹，右侧选择变更的文件夹，根据应用请选择“/page”、“/comfig”的父级文件夹

    2.点击右上角查看本地文件夹比对结果，查看汇总的右侧文件夹与左侧文件夹多出或者差异的文件

    3.比对结果左边灰色表示不存在的文件，蓝色表示存在但是有变动的文件。右边蓝色表示存在且有变动的文件，红色表示不存在的文件，绿色表示多出的文件

    4.通过录入差异文件列表导入excel或者手动录入差异文件目录，最后在比对结果页面查看本页面比对数据与导入数据的差异。
