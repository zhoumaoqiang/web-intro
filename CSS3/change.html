伪元素、伪类选择器

列表的自定义标号~list-style-type的值：decimal（阿拉伯数字）、disc（实心圆）、circle（空心圆）、square（实心方块）、lower-roman（小写罗马数字）、upper-roman（大写罗马数字）、lower-alpha（小写英文字母）、upper-alpha（大写英文字母）、none（无符号）、cjk-ideographic（浅白的表意数字）、georgian（传统的乔治数字）、lower-greek（基本的希腊小写数字）、hiragana（日本平假名字符）、hiragana-iroha（日本平假名序号）、katakana（日本片假名字符）、katakana-iroha（日本片假名序号）、lower-latin（小写拉丁字母）、upper-latin（大写拉丁字母）。

使用自增的编号，采用属性：counter-increment: 计数器的名字; counter-reset: 计数器的名字; 可以给与伪元素content的值为counter(计数器的名字, 列表样式)。

结构伪类选择器：root、first-child、last-child、nth-child(n)、nth-last-child(n)、only-child、first-of-type、last-of-ttype、nth-of-type(n)、nth-last-of-type(n)、only-of-type(n)、empty。

UI元素伪类选择器：link、visited、active、hover、focus、enabled、disabled、checked、default、read-only、read-write、::selection。最后一个前面有两个冒号，表示被选中的元素，firefox支持。

特殊的伪类选择器：not、target。

浏览器CSS前缀：-ms-(IE)、-moz-(firefox)、-o-(opera)、-webkit-(chrome、safari)

字体和文本

  • 字体相关属性

  • font：复合属性，包括font-style、font-variant、font-weight、font-size、line-height、font-family的复合属性值。
  • color：字体颜色，值为颜色名、十六进制颜色值、rgb()、rgba()、hsl()、hsla()
  • font-family：设置文字的字体，需要浏览器内嵌字体的支持，或者引入字体，多个字体以优先找到的字体为准。
  • font-size：xx-small、x-small、small、medium、large、x-large、xx-large、larger、smaller、直接设置大小。
  • font-size-adjust：对不同字体的字体尺寸进行微调。
  • font-stretch：字体横向拉伸，默认为normal，可以设置narrower或者wider。
  • font-style：字体风格，normal、italic、oblique（正常、斜体、倾斜）。
  • font-weight：字体粗细，值为100-900、lighter、normal、bold、bolder。
  • text-derection：装饰线，none、blink、underline、line-through、overline（无、闪烁、下划线、删除线、上划线）。
  • font-variant：字体大写字母的格式，normal、small-caps（正常，小型的大写字母）。
  • text-shadow：字体阴影。一组值为color、xoffset、yoffset、length / xoffset、yoffset、radius、color，同组的值使用空格分隔。可以给文字添加多组阴影，每组使用逗号分隔。
  • text-transform：字体的大小写，值为none、capitalize、uppercase、lowercase。
  • line-height：行高，负值的行高可以展示出阴影效果，可用于控制IFC。
  • letter-spacing：字符之间的间隔，值为normal或者具体的数值。
  • word-spacing：单词之间的间隔。

  • 文本相关属性

  • text-indent：设置段落文本的缩进。
  • text-overflow：文本溢出的控制，值为clip或者ellipsis，分别表示裁剪和自动缩减成...，前提是该元素必须要设置为overflow:hidden，否则文本会溢出正常显示。
  • vertical-align：元素的垂直对齐方式，auto、baseline、sub、super、top、middle、bottom、length。
  • text-align：文本水平对齐，left、center、right、justify。
  • direction：文本流入方向，ltr、rtl。
  • word-break：语句的换行方式，normal、keep-all(只能空格和连字符)、break-all(允许单词中)。
  • white-space：对空格的处理方式，normal(边界自动换行)、nowrap(不换行)。
  • word-wrap：文本的换行方式，normal、break-word(允许单词中间换行)。

定义字体： 

          @font-face {
            font-family: fontname;
            src: local(localfont), url(url) format(fontformat);
            sRules
          }
sRules代表字体样式的设置，如粗体、斜体等这样同一个fontname可以对应正常、粗体、斜体，就不用在使用的时候单独标注字体。src对应local部分可以不写，写上是表明优先使用本地的字体。

背景、边框、边距

  • 背景相关属性

  • background: 复合属性。
  • background-color：背景颜色，没有背景图片时显示的背景颜色。
  • background-image：背景图像。值为url()。
  • background-position：值为x、y坐标位置，可以是具体的距离或者是百分比，top、center、bottom/left、center、right。
  • background-repeat：repeat、repeat-x、repeat-y、no-repeat。
  • background-attachment：scroll、fixed。
  • background-clip：border-box、padding-box、content-box。
  • background-origin：padding-box、border-box、content-box。
  • background-size：contain、cover、百分比、具体尺寸。

  • 边框

  • border：这是一个复合属性，包括边框的样式、粗细和颜色。
  • border-color：边框颜色，顺序为上右下左。
  • border-style：边框样式，顺序上右下左，值有none、dashed、dotted、solid、hidden、double、groove、ridge、inset、outset。
  • border-width：边框粗细，顺序也是上右下左。
  • 渐变边框：border-top-colors、border-right-colors、border-bottom-colors、border-left-colors，值为多个颜色值。
  • border-radius：圆角边框，顺序为左上角、右上角、右下角、左下角。
  • border-image：图片边框，值为如下四部，url() sliceNumber/widthNumber(px) repeat。
  • border-image-source：图片边框的地址。
  • border-image-slice：值为1-4位，表示上右下左边框的切割值。
  • border-image-width：值为1-4位，表示边框上右下左的宽度。
  • border-image-repeat：边框图片的覆盖方式，stretch（拉伸）、repeat（平铺）、round（取整），可取两只值表示横向和纵向。

大小、定位、轮廓

决定元素大小的属性：height、min-height、max-height、width、min-width、max-width。

box-sizing：控制盒模型的计算方式，包括content-box、padding-box、border-box。

resize：窗口调节是对元素的控制，值有none、both、horizontal、vertical、inherit。

postion：定位，absolute（绝对定位）、relative（相对定位）、fixed（固定定位）、static（静态）。搭配left、right和top、bottom属性与参考系进行元素位置的确定。

z-index：元素漂浮层的层序，只有当定位为非静态有效。

outline：外边框，属性与border类似。

盒模型与布局

  • 布局

  • float：浮动的元素相当于display被设置为block，这些浮动的块组件会紧贴并靠近边框。值有left、right。
  • clear：去除浮动，值有none、left、right、both。
  • clip：对元素进行裁剪，值为auto或者rect函数。
  • overflow：对元素溢出部分的处理，值为visible、auto、hidden、scroll。
  • overflow-x和overflow-y：表示x、y方向溢出的处理。
  • visibility和display。

    • 表格中display的值

    • table：表格
    • table-caption：表格标题
    • table-cell：表格单元格
    • table-column
    • table-column-group
    • table-header-group
    • table-footer-group
    • table-row
    • table-row-group
  • box-shadow

  • 分列效果

  • columns：包括column-width和column-count，指定列的宽度和数量。
  • column-rule：复合属性，包括column-rule-width、column-rule-style和column-rule-color，列的分割线的线型包括none、dotted、dashed、solid、double、groove、ridge、inset、outset。
  • column-gap：指定长度为各列之间的间距。
  • column-fill：控制各列的高度，值为auto或者balance（以最高的列为准）。

表格、列表、光标

  • 表格样式的相关属性

  • border-collapse：控制表格两个单元格是合并还是分开的，即应用边框的时候是两条边框还是一条，值为seperate或者collapse。
  • border-spacing：当单元格分开的时候，分开的间距。
  • caption-side：用于设置表格标题的位置，默认为顶部，值为top、right、left、bottom。
  • empty-cells也是单元格分开时生效，指单元格没有内容时是否显示边框，值为none或者show。
  • table-layout：表格的宽度适配方式，值为auto或者fixed。
